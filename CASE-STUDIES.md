# Portfolio Case Studies

These case studies are fictionalized and sanitized. They are designed to demonstrate thinking patterns, not disclose internal company information.

## Case Study 1: Security Architecture in a Regulated Environment

| Item | Description |
| --- | --- |
| Business problem | A regulated business needs secure technology delivery with traceable controls |
| Approach | Security by Design, risk classification, control mapping and security opinion workflow |
| Evidence | Risk matrix, architecture decision records, control checklist and executive summary |
| Recruiter signal | Demonstrates senior security architecture and governance communication |

## Case Study 2: Secure Cloud Foundation

| Item | Description |
| --- | --- |
